# POLYFILL.js
UPPERCASE 기반 프로젝트에서 구버전 브라우저 지원을 해야할 때 사용하는 Polyfill 스크립트

1. [babel-cli](https://www.npmjs.com/package/babel-cli)를 설치합니다.
2. `node_modules` 폴더와 `.babelrc`를 복사합니다.
3. 이후 아래 명령을 수행합니다.
```
babel __SCRIPT -o __SCRIPT_ES5
```
